Здравствуйте! Программа работает следующим образом:
1)После имени исполняемого файла указываете флаги(--forward, --reverse, -- file). В случае файла указать так же после --file флаг операции
(--forward, --reverse), затем в кавычках имя файла, который хотите открыть, например, "filename.txt"
2)Примеры аргументов для "--forward": 1+2+3 (как видите никаких пробелов не нужно)
3) Примеры агрументов "--reverse": 3 4 + =(Программа остановится, когда увидит =, каждый символ должен иметь по пробелу с каждой стороны, кроме первого и последнего)
4) С --file аналогично
5)Немного о программе.
Мой код очень трепетно относится к выделенной памяти, поэтому не советую что-то менять, это произошло из-за того,что
в самый последний момент мне пришлось изменить поток ввода, поэтому char* str  не мог сохранить нормально переменную.
В связи с вышеописанным фактом, ввод выглядит страшнее, чем обычно) 
get() - возвращает символ и переходит на следющий
peek() - смотрит текущий символ
number() - собирает число
fac() - смотрит, чтобы number не взял лишнего и расставляет приоритеты в --forward
term()- смотрит, чтобы сначала выполнялось умножение
infix() - считает инфиксную форму записи алгебраического выражения
opn() - считает постфиксную форму записи алгебраического выражения
6) В программе есть недочет, связанный с делением на 0. В инфиксной форме это никак не обрабатывается мною, но система сама выводит ошибку и не исполняется, поэтому можно простить(пожалуйста... )
